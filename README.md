## Package Installation
### Arch Linux
```bash
sudo pacman -S libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev
```

### Debian
```bash
sudo apt-get install libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev
```
